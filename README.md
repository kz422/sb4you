## SB4YOU

SB4YOUは、某カフェのフラペチーノのカスタマイズを楽しむためのサイトです。

## Feature

- フラペチーノ一覧から特定のフラペチーノを選択すると、カスタマイズが確認できます。

- Choose frapecino from index,now you can see how to sustomize
