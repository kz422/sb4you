## Feature

- 匿名にてメッセージの書き込み  
- カテゴリごとの書き込み表示
- 書き込みは24時間で自動的に削除  
- 自身の書き込みは手動でも削除可能  
- ダークモードのオンオフ

- Post anonymously  
- Index by post category
- Post will be deleted automatically after 24h
- Own post can be delete by manually
- Switch dark-mode

## Usage

1. Push [+]
2. Enter name,title,comment
3. Just post!!
